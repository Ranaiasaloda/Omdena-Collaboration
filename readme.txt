# Healthcare Tweets Sentiment Analysis

This project analyzes tweets about healthcare and classifies them into three sentiments: positive, negative, and neutral.

## Project Structure

- `data/` : contains the CSV file with tweets
- `src/` : Jupyter notebook with analysis and notes
- `plots/` : all generated plots and WordCloud
- `README.md` : project description and instructions

## How to Run

1. Open `src/tweets.ipynb` in Jupyter Notebook.
2. Run all cells to generate plots and WordCloud.
